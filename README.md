# Harmandeep Kaur Portfolio Website

This is a responsive portfolio website built with HTML, CSS, and JavaScript.

## How to use

1. Open `index.html` in your browser.
2. Replace project GitHub links with exact repository links.
3. Add your resume PDF named `Harmandeep_Kaur_Resume.pdf` in the same folder.
4. Push this folder to GitHub.
5. Enable GitHub Pages from repository settings.

## Suggested customizations

- Replace the project links with exact GitHub repositories.
- Add project screenshots if you want.
- Update dates or contact details if needed.
- Add a professional photo only if you want; the current design does not require one.
